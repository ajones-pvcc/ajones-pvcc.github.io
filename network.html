Chapter 4 terms:
Term Definition
Network classifications
LAN local area network. consists of a collection of computers in a single building or building complex
MAN metropolitan area network. a network of intermediate size
WAN wide area network. links machines over a greater distance
Network Models
bus network a network topology in which nodes are directly connected to a common half-duplex link called bus
star network an implementation of a spoke-hub distribution paradigm in computer networks
Protocols
CSMA/CD Ethernet carrier sense, multiple acces with collision detection. dictates that each messege be broadcast to all machines on the bus
CSMA/CA Wifi carrier sense multiple access with collision avoidance. a protocol for carrier transmission in 802.11 networks
Combining networks:
repeater passes signals back and forth between the two original buses without considering the meaning of the signals
bridge looks at the destination address that accompanies each messege and forwards a messege across the connection only when that messege is destined for a computer on the other side
switch a bridge with multiple connections
router special purpose computers used for forwarding messeges
gateway the "point" in which one network is linked to an internet
Client/server model
p2p: peer-to-peer processes that provide service to and recive service from each other
distributed system consists of software units that execute as processes on different computers
cloud computing huge pools of shared computers on the network can be allocated for use by clients as needed
Internet Addressing
Tier 1 consists of very high-speed, high-capacity, international WANs
Tier 2 more regional in scope and less potent in their capabilities
Tier 3 engaged in delivering local access to the internet
IP address
ICANN internet coporation for assigned names and numbers. a nonprofit corporation established to coodinate the internet's operation
DNS domain name system. an internet-wide directory system
TLD: Top-level domain the last segment of a domain name
Country-code TLDs two-letter TLDs for each specific country
Internet protocols
FTP file transfer protocol. an application for listing and copying files across the network
HTTP hyper text transfer protocol. the set of rules for transferring files
SMTP simple mail transfer protocol. defines a way that two computers on the network may interact when transmitting an email message
World Wide Web
URL uniform resource locator. locates and retrieves documents
client-side activities additional activity performed by a client
server-side activites additional activity performed by a server
Internet four-layer software
application layer software units such as clients and servers that use internet communication
transport layer used to send and receive messeges over the internet
network layer decides in which direction a packet should be sent
link layer the resbonsibility of transffering the packet
Internet is:
packet-switching network the transfer of small pieces of data across various networks
fault tolerant a quality of a computer system that gracefully handles the failure hardware and software
redundant duplicate devices that are used for backup purposes
Cybersecurity
malware attacks incorporated with the use of malicious software
virus software that infects a computer by inserting itself into programs that already reside in the machine
worm an autonomous program that transfers itself through a network
trojan horse a program that enters a computer system disguised as a desirable program
spyware another term for malicious software
phishing a technique of obtaining information explicitly by simply asking for it
DDoS attack denial of service attack. a network attacked by software being executed on other computers in the system
Protection
firewall filters out traffic pasing through a point in the network
spam filter a specfic variation of firewalls designed to block unwanted emails
proxy server a software unit that acts as an intermediary between a client and a server
antivirus software used to detect and remove the persense of known viruses and other infections
cryptography the study of sending and receiving messages securely in the persence of adversaries
public key encryption involves techniques in which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure
