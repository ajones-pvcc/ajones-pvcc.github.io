Operating Systems Terminology
Term Definition
operating system the software that controls the overall operation of a computer
job queue storage organization in which objects are ordered in first-in first-out
FIFO first in-, fist-out. objects are removed from a queue in the order in which they arrived
interactive processing a program to carry on a dialogue with the user through remote terminals
time-sharing operating systems that provide service to multiple users at the same time
multiprogramming time is divided into intervals and then the execution of each job is restricted to one interval at a time
application software programs that perform tasks particular to the machine's utilization
system software performs tasks that are common to computer systems in general
utility software programs that perform activites that are fundamental to computer installations
graphical user interface (GUI) a more modern version of shells
shell an older user interface that communicated with users through textual messages using a keyboard and monitor screen
kernal contains software components that perform the very basic functions required by the computer installation
device driver software units that communicate with the controllers
file manager coordinates the use of a machines mass storage facilities
memory manager coordinates a machines use of main memory
virtual memory a large "fictional" memory space created by paging
boot strapping transfers the operating system from mass storage into main memory
ROM read-only memory. its contents can be read but not altered
deadlock two or more processes are blocked from progressing
spooling holding data for output at a later but more convenient time
super user (or administrator) accounts are established by a person known as the super user/administrator
multi-factor authentication to login into an account you need more than just a password. Such as an access code and/or finger print
