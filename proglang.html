HTML: Chapter 6: Programming Languages

Term Definition
machine language a set of coded instructions created and designed for a specific computer to use
debugging the process of locating and fixing or bypassing bugs (errors) in computer program code
assembler a program that converts assembly language into machine code
assembly language a programming language that consists of instructions that are mnemonic codes for corresponding machine language instructions
translator a programming language processor that converts a computer program from one language to another
compiler system software that converts source code written in a programming language into another computer language
interpreter a program that directly executes the instructions in a high-level language, without converting it into machine code
cross-platform software a product or system that can work across multiple types of platforms or operating environments
natural language a subset of artificial intelligence, computer science, and linguistics-focused on making human communication
formal language a language with mathematically precise construction rules
machine independence programs and procedures which function in essentially the same manner regardless of the machine on which they are carried out
high-level programming language close to natural language spoken and written by humans
object-oriented programming (OOP) used to structure a software program into simple, reusable pieces of code blueprints
scripting language a computer language intended to solve relatively small programming problems
integer used to represent real numbers that do not have fractional values
float a data type built into the compiler that's used to define numeric values with floating decimal points
string a data type used in programming, such as an integer and floating point unit, but is used to represent text rather than numbers
primitive data type a data type that is built into a programming language
data structure way in which data are stored for efficient search and retrieval
python list data structure used to store multiple items in a single variable
python tuple data structure used to store multiple items in a single variable
python set data structure used to store multiple items in a single variable
python dictionary data structure used to store data values in key:value pairs
function a unit of code that is often defined by its role within a greater code structure
local variable a variable that can be accessed only from within the loop or procedure in which it was created
global variable variables that can be accessed by any part of the program
source program the fundamental component of a computer program that is created by a programmer
object program is a computer program translated from the equivalent source program into machine language by the compiler or assembler
